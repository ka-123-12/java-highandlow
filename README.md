# High & Low Game (Java版)

Javaで作成したコンソールまたはGUI（Swing）で遊べるハイアンドローゲームです。

---

## ゲーム内容
- ランダムに生成されたカードの数値を予想
- 「次のカードは今のカードより高いか低いか」をhまたはlにて選択
- 予想が当たればげーう続行可能、外れればゲーム終了
- コンソールでプレイ可能

## 工夫した点
- **ランダム生成ロジック**：カードが公平に選ばれるように設計
- **MVCパターン採用**：ロジックと画面表示を分離して保守性向上
- **入力チェック**：不正入力や例外処理に対応

---

## 技術スタック
- Java 20以上
- Swing（GUI版） / コンソールI/O
- MVC設計

---

## 実行方法

### コマンドライン版
1. リポジトリをクローン
git clone https://github.com/username/highlow-java.git
2. コンパイル
javac -d out src/*.java
3. 実行
java -cp out Main

---

### GUI版
- `Main` クラスを実行するとウィンドウが表示されます
